## Что такое проект?

Проект — это отдельное приложение в Makbool. У каждого проекта свой API Key, своя история ошибок и свои настройки уведомлений.

Примеры: фронтенд-приложение, мобильное приложение, бэкенд-сервис — каждый лучше держать в отдельном проекте.

## Создание проекта

1. Перейдите в [дашборд](https://makbool.ru/dashboard)
2. Нажмите **«Новый проект»**
3. Введите название
4. Нажмите **«Создать»**

После создания вы попадёте на страницу проекта.

## API Key

API Key — это идентификатор вашего проекта. Он передаётся в SDK как `projectKey`.

**Где найти:** Настройки проекта → раздел «API Key».

```
pk_live_a1b2c3d4e5f6...
```

### Ротация ключа

Если API Key скомпрометирован, его можно сменить:

1. Зайдите в **Настройки проекта**
2. Нажмите **«Сменить ключ»**
3. Подтвердите действие

<Warning>
  После смены ключа старый перестаёт работать немедленно. Обновите `projectKey` во всех SDK-интеграциях и задеплойте приложение.
</Warning>

## Настройки проекта

В настройках проекта вы можете:

- **Переименовать проект** — название отображается в дашборде и уведомлениях
- **Сменить API Key** — ротация при компрометации
- **Настроить алерты** — Telegram, email, webhook
- **Загрузить source maps** — для разворачивания минифицированных стектрейсов
- **Управлять командой** — приглашение участников

## Лимит проектов

Количество проектов зависит от плана:

| Plan | Проектов |
|------|:--------:|
| Free | 1 |
| Starter | 3 |
| Pro | 15 |
| Business | Без лимита |

При достижении лимита кнопка **«Новый проект»** станет недоступна. Обновите план или удалите неиспользуемые проекты.

## Удаление проекта

1. Настройки проекта → **«Удалить проект»**
2. Введите название проекта для подтверждения
3. Нажмите **«Удалить»**

<Warning>
  Удаление необратимо. Все ошибки, настройки и аналитика будут удалены безвозвратно.
</Warning>
